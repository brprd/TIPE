# TIPE
Les fichiers utiles sont dans le dossier tipe.
Ce dossier est à télécharger dans son intégralité puisque les programmes qu'il contient sont dépendants.

Certains programmes recquièrent la modification du chemin d'accès au fichier base de donnée. Ce chemin est à spécifier par la variable "input_file".
Afin que les dépendances soient chargées correctement, il est nécessaire d'exécuter un programme en temps que script (Ctrl+Maj+E sous Pyzo).
